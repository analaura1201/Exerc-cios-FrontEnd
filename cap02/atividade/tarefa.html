    Quem criou o JavaScript e em que ano ele foi criado?

O JavaScript foi criado por Brendan Eich em 1995, enquanto ele trabalhava na Netscape Communications Corporation. Inicialmente, o JavaScript foi chamado de Mocha, depois foi renomeado para LiveScript e, por fim, recebeu o nome de JavaScript para aproveitar a popularidade do Java, embora as duas linguagens sejam bem diferentes. O objetivo do JavaScript era adicionar interatividade às páginas web, permitindo a execução de scripts no lado do cliente diretamente nos navegadores.

    O que são eventos em JavaScript? Dê um exemplo simples.

Em JavaScript, eventos são ações ou ocorrências que acontecem no navegador, como um clique de um botão, o movimento do mouse, o carregamento de uma página ou até mesmo o pressionamento de uma tecla. Esses eventos podem ser capturados e tratados usando JavaScript para executar ações específicas quando eles ocorrerem.

    O que é Visual Studio Code?

O Visual Studio Code (VS Code) é um editor de código-fonte gratuito e leve, desenvolvido pela Microsoft. Ele suporta várias linguagens de programação e pode ser altamente personalizado com extensões. Entre suas principais funcionalidades estão autocompletar código, integração com Git, depuração, terminal integrado e suporte a múltiplas linguagens. É ideal para desenvolvedores que buscam uma ferramenta rápida, poderosa e fácil de usar.

    O que é Bootstrap?

Bootstrap é um framework front-end gratuito e open-source, que facilita o desenvolvimento de sites responsivos e com design consistente. Ele oferece componentes prontos (como botões e formulários), um sistema de grid para layouts e estilos CSS prontos para uso, permitindo criar sites de forma rápida e fácil.
